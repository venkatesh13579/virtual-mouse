# 🖱️ AI Virtual Mouse

Control your computer with **hand gestures** using **OpenCV**, **MediaPipe**, and **Python**.  
The **AI Virtual Mouse** replaces traditional hardware with a **gesture-based interface**, making interaction more intuitive and futuristic.

---

## ✨ Features
- 🖐️ **Cursor Movement** – Move the mouse using your index finger  
- 👆 **Left Click** – Index + Middle finger gesture  
- 🤟 **Right Click** – Index + Middle + Ring fingers  
- 🔼 **Scroll Up** – All fingers extended  
- 🔽 **Scroll Down** – Thumb + Pinky gesture  
- ⏭️ **Next Slide / Right Arrow** – Thumb extended  
- ⏮️ **Previous Slide / Left Arrow** – Pinky extended  
- 🔊 **Volume Control** – Hand gestures for volume up/down  
- ⚡ **FPS Counter** – See real-time performance
